# Welcome to CamelMind

CamelMind is an open-source documentation platform built on Next.js. Write docs in Markdown, configure navigation in YAML, and publish a beautiful site in minutes.

<Callout type="tip">
Your documentation journey starts here. Add Markdown files to the `content/` folder and define your nav in `nav/nav.yml`.
</Callout>

## Why CamelMind?

Most documentation platforms force teams to choose between flexibility and ease of use. CamelMind gives you both:

- **Markdown-first** authoring with MDX components
- **YAML navigation** decoupled from file paths
- **Built-in search** across all docs
- **Version support** for multi-release documentation
- **Optional SSO + RBAC** for internal and public docs on the same site
- **Static export** for offline or self-hosted deployments
