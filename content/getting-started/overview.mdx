Game Warden is a secure platform for DoD software deployment. This guide covers the essentials.

<Callout type="tip">
  You can access Game Warden from NIPRNet or through approved access pathways.
</Callout>

## What is Game Warden?

Game Warden provides a FedRAMP and IL4/IL5-ready environment for deploying containerized applications.

## Key Concepts

- **App Central** — the main interface for vendors to submit and manage applications
- **Body of Evidence (BoE)** — the documentation package required for ATO
- **Findings** — security issues surfaced during pipeline scans
