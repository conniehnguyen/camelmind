## v3.13.0 — May 20, 2026

<Callout type="important">
  **Breaking Change**: The `gw-standard` storage class now uses GP3 EBS volumes instead of GP2. Existing PVCs are not affected. New PVC provisioning requests using `gw-standard` will receive GP3 volumes, which have different IOPS baseline characteristics. Review your application's storage I/O requirements if you are provisioning new volumes.
</Callout>

### New Features

- **Jaeger Distributed Tracing (GA)**: Distributed tracing via Jaeger is now generally available for all environments. Enable tracing for your namespace by submitting a support ticket requesting Jaeger integration. Trace data is retained for 7 days.
- **FedRAMP POA&M Dashboard**: A new dedicated POA&M dashboard is available for FedRAMP path tenants, showing all open accepted risks, their target remediation dates, and days remaining. Access under **Applications > [App] > BoE > POA&M**.
- **API Key Scoping**: Organization API keys can now be scoped to specific applications in addition to organization-wide access. Useful for restricting CI/CD pipeline keys to a single application's registry namespace.

### Improvements

- Upgraded Kubernetes cluster version to 1.30 across all environments. Tenants should verify their Kubernetes API version compatibility. The Kubernetes 1.27 API deprecations apply; deprecated API versions will return warnings in pipeline logs.
- Semgrep SAST ruleset updated to include 47 new rules targeting misconfigured Kubernetes RBAC patterns and Helm chart security issues.
- App Central onboarding status page now shows estimated days remaining for each active onboarding phase based on historical data.
- Harbor vulnerability scan reports can now be exported as PDF from the image detail page.

### Bug Fixes

- Fixed an issue where Org Admin role changes did not immediately reflect in the portal navigation without a full page reload.
- Resolved incorrect SLA countdown display for findings detected on weekends (SLA timer was counting calendar days instead of business days for Medium/Low severity).
- Fixed a bug where the BoE "Submit for Review" button was not visible when the browser zoom level was set above 125%.

## v3.12.0 — May 6, 2026

### New Features

- **Managed PostgreSQL 15 GA**: PostgreSQL 15 is now generally available as a managed service option. New database instances default to PG15. Existing PG14 instances are not automatically upgraded; contact support to schedule an upgrade.
- **pgvector Extension Support**: Managed PostgreSQL instances now support the `pgvector` extension for storing and querying vector embeddings. Useful for AI/ML workloads using semantic search or RAG patterns. Request `pgvector` enablement via a support ticket.

### Improvements

- Loki log retention for DoW IL5 environments extended to 3 years, from the previous 2-year retention period, to meet updated DISA audit log requirements.
- The GW CLI (`gw`) now includes a `gw pipeline status` command that outputs the current pipeline run status and scan results for a given image tag.

### Bug Fixes

- Fixed an issue where deleting an application in App Central did not remove the associated Harbor project, causing orphaned registry storage.
- Resolved a Vault Agent injection failure that occurred when pod names contained uppercase letters.
