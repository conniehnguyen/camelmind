# Quickstart

## Prerequisites

- A Camel Trading Company account (v2 plan or higher)
- An API key from your [Merchant Dashboard](https://dashboard.cameltrading.example)
- A Partner API key if you're accessing the Partner Network API

## Make your first request

Fetch your caravans:

```bash
curl https://api.cameltrading.example/v1/caravans \
  -H "Authorization: Bearer <your-token>"
```

Fetch shared routes from the Partner Network:

```bash
curl https://partners.cameltrading.example/v2/routes \
  -H "X-Partner-Key: <your-partner-key>"
```
