# v2 Overview

Version 2 of the Camel Trading Company platform introduces the **Partner Network** — a shared infrastructure layer that lets certified trade partners exchange route intelligence, register oasis waypoints, and receive real-time caravan events.

## What's new

- **Partner Network API** — A dedicated API for certified partners to publish and consume shared route data
- **Oasis Waypoint Registry** — A crowd-sourced map of safe stopping points across all partner-operated routes
- **Real-time Events** — Subscribe to caravan arrival and departure events across the network

## API Reference

This version exposes two APIs:

- **Camel Trading Company API** — The core platform API for managing caravans, cargo, and bazaar orders
- **Partner Network API** — The partner-facing API for shared routes, waypoints, and events

Use the tab switcher in the API Reference sidebar to switch between them.
