The User Management panel allows admins to manage access across the platform.

<Callout type="warning">
  Changes to user roles take effect immediately. Use caution when modifying access.
</Callout>

## Actions

- **Invite user** — send an email invitation to a new user
- **Assign roles** — grant vendor, admin, or read-only access
- **Suspend account** — temporarily revoke access without deleting the account
- **Delete account** — permanently remove a user
