App Central is the primary interface for managing your applications on Game Warden. From App Central, you can register new applications, track onboarding status, manage your Body of Evidence (BoE) package, view security findings, and submit support tickets.

![App Central main dashboard showing the applications list with onboarding status and findings summary](/images/gwa-app-central-main.png)

## What Is App Central?

App Central is the vendor-facing dashboard within the Game Warden portal. It serves as the single pane of glass for all activities related to your application's lifecycle on the platform—from initial onboarding through production deployment and ongoing compliance maintenance.

## Key Sections

### Applications List

The Applications List is the home view of App Central. It shows all applications registered under your organization, with summary cards displaying:
- Application name and version
- Current deployment path (DoW, FedRAMP, or Commercial)
- Onboarding status (e.g., Entrance Criteria Review, Security Review, Deployed)
- Open findings count by severity
- BoE completion percentage

Click any application card to open its detail view.

### Body of Evidence (BoE) Status

The BoE tab within an application's detail view shows the completion status of each required BoE section. Sections are marked as Not Started, In Progress, Under Review, or Complete. You can upload documents, add comments, and track reviewer feedback from this tab.

See [Body of Evidence](/gw-app/boe) for detailed instructions.

### Findings

The Findings tab shows all security findings associated with your application—from pipeline scans, manual security review, or continuous monitoring. Findings are grouped by severity (Critical, High, Medium, Low) and show current status (Open, In Remediation, Accepted Risk, Resolved, False Positive).

See [Findings](/gw-app/findings) for details on remediating findings.

### Notifications

The Notifications panel (bell icon, top right) surfaces important events:
- New findings detected
- BoE section review feedback
- Pipeline scan completions
- Support ticket updates
- ATO status changes

You can configure email notification preferences under **Profile > Notification Settings**.

## Navigating App Central

| Action | Where to Find It |
|---|---|
| Register a new application | App Central home > **New Application** button |
| Upload BoE documents | Application detail > **BoE** tab > section > **Upload Document** |
| View pipeline scan results | Application detail > **Pipeline** tab |
| View and remediate findings | Application detail > **Findings** tab |
| Submit a support ticket | Top navigation > **Support** > **New Ticket** |
| Manage team members | **Organization Settings** > **Members** |
| View audit logs | **Organization Settings** > **Audit Log** |

## Quick Actions

From any application card on the Applications List, you can access quick actions via the three-dot menu:

- **View Details** — Opens the full application detail view
- **New Pipeline Run** — Triggers a manual pipeline scan with the latest images in the registry
- **Create Support Ticket** — Opens a pre-filled support ticket with the application context
- **Archive Application** — Archives the application (cannot be undone without contacting support)

<Callout type="tip">
  If you have multiple applications, use the search bar and filter controls at the top of the Applications List to quickly find what you need. You can filter by deployment path, onboarding status, and open findings severity.
</Callout>
