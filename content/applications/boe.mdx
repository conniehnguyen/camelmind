The Body of Evidence (BoE) is a documentation package that supports your ATO process.

## Required Sections

- System Security Plan (SSP)
- Continuous Monitoring Plan
- Incident Response Plan
- Configuration Management Plan

<Callout type="important">
  All sections must be completed before your application can advance to the review stage.
</Callout>
